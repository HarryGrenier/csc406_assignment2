Detailed Description

Base class for 2D graphic objects with position, orientation, and scale properties.

This abstract class defines a 2D object in a coordinate system, allowing the object to have a position, an orientation (rotation), and a scale. Derived classes must implement the draw method to handle the specific rendering of the object.

Note
This is an abstract class, and the draw method must be implemented by derived classes.
