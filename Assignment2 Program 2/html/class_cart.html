Detailed Description

Represents a cart object with wheels, position, orientation, and scale.

This class models a cart that inherits from ComplexGraphicObject2D. It includes properties for position, orientation, and wheel rotation, and provides methods for updating these properties as well as drawing the cart. It also tracks the movement direction of the cart.
