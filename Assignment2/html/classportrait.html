Detailed Description

A class representing a portrait object that can be drawn with various features such as eyes, nose, mouth, etc.

This class inherits from GraphicObject2D and provides functionality to draw a portrait with customizable features. It includes helper methods to draw specific facial features like eyes, nose, and mouth. Additionally, the class allows setting and getting the position, orientation, and scale of the portrait.

See also
GraphicObject2D
Note
This class includes static data members to track portrait instances.
